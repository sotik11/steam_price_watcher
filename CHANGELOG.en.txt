Steam Price Watcher — Changelog
===============================

v0.1.2.2 (2026-06-16)
- Game/card names with HTML entities (e.g. "Assassin&#x27;s") are now
  decoded to normal characters.
- Profile-background rarity (Common/Uncommon/Rare) is moved out of the game
  name into the type ("Uncommon Profile Background"); game name stays clean.

v0.1.2.1 (2026-06-15)
- watch.py no longer logs ERROR when config.json isn't there yet (fresh /
  clean install) — a calm log line instead of red.
- Installer also drops the legacy scheduled task (SteamCardWatch).
- Spare window icon (__app.ico) used if the primary one is missing/broken.

v0.1.2.0 (2026-06-15)
- Backup/restore: "Export" / "Import" buttons in Settings. Export saves
  config + all data into a .zip; import restores them (overwrite + restart).
  Format is interchangeable with the installer.

v0.1.1.0 (2026-06-15)
- "Do not disturb" mode: a time window (__:__ - __:__) during which
  Telegram alerts are not sent (even when antispam has expired). Checked
  in local time, works across midnight. Window start/end are logged.

v0.1.0.1 (2026-06-14)
- Fixed: "Save" in Settings no longer wipes "Bonus content" (the «Games»
  tab), sort order, or column widths.
- Currency symbol in the "Price" and "Minimum" columns on the Games tab.

v0.1.0.0 (2026-06-14) — First tracked release
- App renamed to "Steam Price Watcher".
- "About" tab: version, author, changelog, links.
- Custom window / taskbar icon; version module.

v0.0.6.0 (2026-06-14)
- Column widths persist across restarts + double-click a border to auto-fit.
- Table keyboard shortcuts: Delete, Enter, Space, Esc.
- Funds-on-hold line in the widget; History timestamps in local timezone.

v0.0.5.0 (2026-06-13)
- Installer (Inno Setup) + setup.bat: bundled Python, update /
  reinstall / uninstall modes, user-data import.

v0.0.4.0 (2026-06-11)
- "Games" tab: wishlist price tracking with all-time-low alerts.
- Steam session-expired detection.

v0.0.3.0 (2026-06-08)
- Steam login (manual ID + browser cookies) and wallet balance in the widget.
- "History" tab with CSV export; import active listings & buy orders from Steam.

v0.0.2.0 (2026-06-06)
- Floating user widget; font scaling x1–x5; themes; window geometry
  persistence; bilingual UI (uk/en).

v0.0.1.0 (2026-06-05)
- Core Steam Market card price monitoring; Telegram alerts (buy/sell);
  semi-automatic buying; check scheduler.
