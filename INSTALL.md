# Установка и перенос на другой ПК

Два способа поставить приложение: **инсталлятор** (рекомендуется) и
**ручной батник** (запасной). Оба ставят Python при необходимости,
создают venv, тянут зависимости и кладут ярлык на Рабочий стол.

---

## Способ 1 — инсталлятор (рекомендуется)

Готовый `SteamPriceWatcher-Setup-<версия>.exe` — это обычный мастер
установки. Внутри него уже всё: код, ресурсы и оффлайн-установщик Python.

1. Запусти `.exe`.
2. Выбери папку (по умолчанию `%LOCALAPPDATA%\Programs\SteamPriceWatcher` —
   **не меняй на `Program Files`**: приложение пишет свои данные рядом с
   собой, а туда без админ-прав писать нельзя).
3. Мастер сам: поставит Python 3.13 (если его нет в системе), создаст
   venv, поставит зависимости, сделает ярлык.
4. Запускай с ярлыка на Рабочем столе.

> **Нужен интернет.** На этапе настройки окружения `pip` качает
> зависимости (ttkbootstrap, requests, pillow и т.д.) с PyPI. Без сети
> venv создастся, но без пакетов — приложение не стартует.

> **Если приложение не запускается / ярлык ругается на `pythonw.exe`** —
> значит venv не собрался. Смотри лог `setup_env.log` в папке программы:
> там видно, нашёлся ли Python и на чём упал `pip`. Можно дособрать
> вручную: запусти `setup_env.bat` из папки программы.

### Повторный запуск инсталлятора
Если программа уже установлена, мастер на первой странице предложит:
- **Оновити** — поставить поверх, сохранив данные и задачу планировщика;
- **Перевстановити з нуля** — снести данные/venv/задачу и поставить заново;
- **Видалити, зберігши дані** — скопировать данные на Рабочий стол
  (папка `Steam Price Watcher`) и полностью удалить;
- **Повністю видалити** — снести всё.

### Удаление
Через «Параметры → Приложения» или `unins000.exe` в папке программы.
Удаление: снимет задачу из Планировщика Windows, снесёт программу и venv,
спросит — удалять ли сохранённые данные и удалять ли Python.

## Способ 2 — вручную (без инсталлятора)

Если переносишь копированием папки:

1. Скопируй папку проекта на новый ПК (**куда угодно, на любой диск** —
   пути относительные). **Не копируй** `.venv` — он непереносимый,
   создастся заново.
2. Запусти `setup.bat` — он поставит Python (если нет), создаст venv,
   поставит зависимости, сделает ярлык.
3. Запускай с ярлыка или через `gui.bat`.

---

## После установки — планировщик

Фоновая проверка цен (`watch.py`) запускается через Планировщик Windows.
Задача привязана к конкретному пути, поэтому **на новом ПК её надо
создать заново**: открой приложение → вкладка **Планувальник** →
**Створити завдання**. Старая задача на прежнем ПК останется — удали её
там, если нужно.

---

## Про вход в Steam на новом ПК

`config.json` переносит токен Telegram и cookies. Cookies к железу не
привязаны, но **протухают**. Когда протухнут — для повторного логина
нужен браузер с залогиненным Steam **на этом ПК** (кнопка «Імпорт з
браузера» читает cookies локального браузера).

## Пересборка инсталлятора (для разработки)

Нужен [Inno Setup 6](https://jrsoftware.org/isdl.php)
(`winget install JRSoftware.InnoSetup`).

```
installer\build_installer.bat
```

Скрипт скачает bundled-Python (кэшируется) и скомпилирует
`installer\dist\SteamPriceWatcher-Setup-<версия>.exe`.

**При выпуске новой версии:** подними `PYVER` в `build_installer.bat` и
`MyAppVersion` + `PyVersion` в `installer.iss` до актуальной
**3.13.x** (не 3.14 — у зависимостей может не быть готовых wheels).
